# Material You Component

Playground dokumentasi Svelte + Astro untuk pola UI Material Design 3 (Material You). Fokus proyek ini adalah menampilkan komponen, token, dan layout dengan struktur yang modular dan rapi.

Repository: `https://github.com/itzkazuri/material-you-component`

Fork from: `https://github.com/hiratazx/material-you-css`

---

## Ringkasan

- Komponen Svelte modular untuk demo dan dokumentasi.
- Theme switcher + dukungan seed color (Material You).
- Halaman Astro untuk dokumentasi statis yang cepat.

---

## Memulai

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

---

## Struktur

- `src/pages` halaman Astro (landing dan docs)
- `src/components/svelte` komponen Svelte untuk demo UI
- `src/assets/app.css` definisi token Material You
