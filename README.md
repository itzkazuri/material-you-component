# Material You Component

A Svelte + Astro documentation playground for Material Design 3 (Material You) UI patterns. This project is focused on showcasing components, tokens, and layouts with a clean, modular structure.

Repository: `https://github.com/itzkazuri/material-you-component`

Fork from: `https://github.com/hiratazx/material-you-css`

---

## Overview

- Modular Svelte components for demos and documentation.
- Theme switcher + seed color support (Material You style).
- Astro pages for fast, static documentation.

---

## Getting Started

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

---

## Structure

- `src/pages` Astro pages (landing and docs)
- `src/components/svelte` Svelte components for UI demos
- `src/assets/app.css` Material You token definitions
